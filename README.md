# MBE 外延结构管理 v1.0

本地单机网页工具，用来记录、编辑、统计和导出 MBE 外延片结构。数据保存在本机 SQLite 数据库里，不需要联网，也不是多人系统。

## 启动

需要先安装 Python 3。

- macOS：双击 `Start MBE Tracker.command`
- Windows：双击 `Start MBE Tracker Windows.bat`

启动后会自动打开浏览器：

```text
http://127.0.0.1:8765
```

使用时保留弹出的终端窗口；关闭窗口或按 `Ctrl+C` 会停止程序。数据库默认保存在：

```text
data/mbe.sqlite
```

第一次启动如果还没有数据库，会自动导入一个样板片 `N260508A`，用于演示普通层、重复层、QD 和 N/P 掺杂。

## 主要功能

- 新建、搜索、复制、删除整片外延片；正式片和测试片可记录样品托编号
- 左侧可切换正式片、测试片和 MBE 参数；测试片可记录 As 压束流比、QD 成岛时间、QD 淀积量、再构温度、QD 生长相对/实际温度、自动计算生长速率、QD 密度、QD 体积中位数、QD 高度、体积 CV、PL 峰位、FWHM 和 PL 强度
- MBE 参数模块按日期记录待机真空度、As 压充盈真空度、As bulk 温度，并可展开记录源炉束流/温度和 As 压参数
- 普通层录入：层名、材料、厚度、掺杂浓度、N/P、温度、QD、备注
- 周期数大于 1 时作为重复层，支持展开编辑子层
- 层、子层和重复层可复制、粘贴、上移、下移、删除和撤回上一步
- QD 层显示但不计入厚度，结构图中用点标记
- 右侧结构图按厚度比例显示，并保留最小显示高度
- 自动统计总厚度、材料厚度、掺杂层、QD 层和有源区掺杂信息
- 快捷项可自定义，也可以把现有层或重复结构加入快捷项
- 导出 JSON、CSV、结构图图片；多片结构图可导出为 PDF
- 全局导出备份会下载一个 `备份日期.zip`，里面包含 JSON、CSV、分类 Excel、正式片概要 Excel 和正式片结构图 PDF

## 导入导出

目前只支持 JSON 导入，不支持 Excel 导入。不同人之间互传数据，建议使用 JSON 文件。

- JSON：用于备份和以后重新导入，最重要
- PDF/图片：用于自己查看、汇报或归档
- CSV：用于简单表格统计

建议经常导出 JSON 做备份，同时导出 PDF 留一份可阅读版本。这个工具还在早期正式版，可能仍有未发现的问题，多留备份更稳妥。

## 发给别人

把对应系统的压缩包发给别人即可。对方解压后双击启动脚本使用：

- macOS 用 `MBE-Tracker-mac-v1.0.zip`
- Windows 用 `MBE-Tracker-windows-v1.0.zip`

如果打不开，优先检查是否安装了 Python 3。
