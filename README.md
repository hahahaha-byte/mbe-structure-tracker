# MBE 外延结构管理

本地单机网页工具，用 SQLite 保存外延片、普通层、重复块和模板数据。第一版使用 Python 标准库后端和原生 HTML/CSS/JS 前端，不需要安装第三方依赖。

## 启动

最简单方式：双击项目目录里的：

```text
Start MBE Tracker.command
```

它会自动启动本地服务并打开浏览器页面。使用时保留弹出的 Terminal 窗口；关闭窗口或在窗口里按 `Ctrl+C` 会停止服务。
如果旧服务已经在运行，再双击一次这个文件会先自动停止旧后端，再用当前最新版代码启动。

也可以手动启动：

```bash
python3 server.py
```

浏览器打开：

```text
http://127.0.0.1:8765
```

数据库默认保存在：

```text
data/mbe.sqlite
```

## 导入现有 Excel

页面左侧可以直接点击 `导入片号N*.xlsx`。也可以命令行导入：

```bash
python3 server.py import-excel
```

## 已支持

- 新建、搜索、编辑、复制整片外延结构
- 添加层默认插在选中层上方；周期数大于 1 时自动作为重复层
- 重复层可展开添加内部层，内部层记录具体材料和厚度
- 删除层后可连续撤回上一步，误删多个也能逐步恢复
- 层或结构块复制、粘贴到选中层上方、上移、下移、删除
- 快捷项可编辑，点击 `+` 可在当前选中层上方插入新层
- 掺杂浓度自由文本，不填或旧表中的 `0` 视为不掺杂
- 生长温度和备注字段
- QD 层可标记为显示但不计入厚度
- 右侧厚度比例结构图、重复块样式、掺杂标记
- 总厚度、材料总厚度、掺杂层数量统计
- JSON/CSV 导出
